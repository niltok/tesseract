### 项目架构

- `src/main/kotlin/goldimax/tesseract`
  - `Forward.kt`: QQ与TG间的消息转发
  - `Picture.kt`: 保存和触发发送图片
  - `Markov.kt`: 利用马尔科夫链构造随机串
  - `ImageMgr.kt`: 图片储存管理器
  - `History.kt`: 消息历史管理器
  - `Connection.kt`: 群连接管理器
  - `Transaction.kt`: 事务管理器
  - `WebPage.kt`: Headless 浏览器控制
